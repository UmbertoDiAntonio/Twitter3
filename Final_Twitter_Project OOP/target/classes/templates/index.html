Mostra questa lista : http://localhost:8080/
Mostra la lista completa dei post: http://localhost:8080/post
Mostra questa lista : http://localhost:8080/setblacklist/(one:two:tree:..)
Mostra i post dell user : http://localhost:8080/post/author/(author)
Mostra i post contenenti il testo: http://localhost:8080/post/author/contains/(message)
Mostra i post con tag : http://localhost:8080/post/author/contains/tag
Mostra i post con retweet : http://localhost:8080/post/author/contains/retweet
Mostra i post dell user con testo: http://localhost:8080/post/author/(author)/contains/(message)
Mostra i post con i parametri temporali :
http://localhost:8080/post/author/(author)/contains/(message)/year/(year)/month/(month)/day/(day)/hour/(hour)
http://localhost:8080/post/author/(author)/contains/(message)/year/(year)/month/(month)/day/(day)
http://localhost:8080/post/author/(author)/contains/(message)/year/(year)/month/(month)
http://localhost:8080/post/author/(author)/contains/(message)/year/(year)

Trucchi :
(message) può essere definito come una lista usando come separatore tra le parole il :
puoi salvare una lista di parole con setblacklist e richiamarla inserendo blacklist in (message)
(author) può essere ignorato, sarà autocompilato come ALL
(message) può essere ignorato, sarà autocompilato come ALL
